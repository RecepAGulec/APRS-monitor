# APRS-monitor
APRS işaretlerinin ekranda gösterilmesi
Devrede APRS işaretlerinin çözülmesi bir Arduino Nano ile yapılmakta ve çözülen işaretler ILI9341 sürücülü bir TFT ekranda gösterilmektedir.
Yazılım, sâbit istasyonlar için düşünülerek tasarlanmıştır. Eğer mobil olarak kullanmak isterseniz, devreye bir GPS işareti uygulamanız ve kodlarda buna uygun değişiklikler yapmanız gerekmektedir.
